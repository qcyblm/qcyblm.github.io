# npm 的安装路径、缓存路径和更换镜像源

# 查看各种路径命令

  • 查看当前 npm 包的全局安装路径
npm prefix -g
1
  • 查看配置列表
npm config ls
1

# 修改路径命令

  • 修改 npm 的包的全局安装路径
npm config set prefix "<fill>"
1
  • 修改 npm 的包的全局 cache 位置
npm config set cache "<fill>"
1

# 修改镜像源命令

  • 查看 npm 当前镜像源
npm config get registry
1
  • 改变 npm 镜像源
npm config set registry "<url>"
1

# yarn 的安装路径、缓存路径和更换镜像源

# 查看各种路径命令

  • 查看 yarn 全局 bin 位置
yarn global bin
1
  • 查看 yarn 全局安装位置
yarn global dir
1
  • 查看 yarn 全局 cache 位置
yarn cache dir
1

# 修改路径命令

  • 改变 yarn 全局 bin 位置
yarn config set prefix "<fill>"
1
  • 改变 yarn 全局安装位置
yarn config  set global-folder "<fill>"
1
  • 改变 yarn 全局 cache 位置
yarn config set cache-folder "<fill>"
1
  • 改变 yarn 全局 link 位置
yarn config set link-folder "<fill>"
1

# 修改镜像源命令

  • 查看 yarn 当前镜像源
yarn config get registry
1
  • 改变 yarn 镜像源
yarn config set registry "<url>"
1
